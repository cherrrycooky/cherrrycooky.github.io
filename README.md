# Yongjae Kim personal website

This is a minimal GitHub Pages / Jekyll personal academic website.

## Edit content

- Home: `index.markdown`
- Papers: `papers.markdown`
- Talks: `talks.markdown`
- Teaching: `teaching.markdown`
- Global settings: `_config.yml`
- Design: `assets/css/style.css`

## Profile photo

Upload your profile photo to:

`assets/img/profile.jpg`

The home page already reserves a photo slot. If `profile.jpg` is missing, it shows a simple `YK` placeholder.

## Migration note

If you previously used the HTML-only version, delete these old files from the repository root:

- `.nojekyll`
- `index.html`
- `research.html`
- `talks.html`
- `teaching.html`
- `styles.css`

The root `cv.pdf` in this package is based on the public CV without phone number.
